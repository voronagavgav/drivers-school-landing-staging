The plan of record for evolving the mature ПДР-prep app (Waves 1–9 done) into the calm,
adaptive, offline-capable, liquid-glass product. **Start with `00`**; docs `01`–`05` are the
detailed layers it synthesizes. Where the layers forked, `00` §6 is canonical.

Delivered via the Mesa harness (`mesa plan` → `run-all` + evaluator + verify gate), then
independently real-browser verified. Content is official-only and honestly counted
(2322 Q · 65 topics · 8 categories · 734 images being restyled, 60 live); ★4.9/testimonials
are placeholders and never ship as real.
